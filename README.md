## Electric Cars Database Management System

A C-based management system for electric vehicle charging stations.

This project simulates a real-world EV charging infrastructure using data structures such as Binary Search Trees, Queues, and dynamic memory allocation.

---

## 🚀 Features

- Manage charging stations and electric vehicles
- Support FAST / MID / SLOW charging ports
- Queue system for unavailable charging ports
- Vehicle and station lookup using Binary Search Trees
- Charging status and payment tracking
- Station statistics and top customer reports
- File-based data persistence
